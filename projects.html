Project 3

I developed a basic 2d-game, making use of classes and objects along with if statements and for loops. The game consists of ball objects which drop from the top of the screen, and theres a rectangular slider object which can be moved left and right using certain keys, and each time the ball hits the slider, the score increments by one.
